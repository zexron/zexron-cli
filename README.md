# zexron-cli
My Easy CLI commands

## Usage
```bash
$ z-cli <command> [options]
```

## Example
```bash
$ z-cli --help
```

## Available Command

### `generate`
To generate a frontend project based on gulp or webpack [learn more](https://github.com/zexron/my-build-tools)

```bash
$ z-cli generate --help

$ z-cli g --help
```
